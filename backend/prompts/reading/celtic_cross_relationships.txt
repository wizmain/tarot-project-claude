## 리딩 요청

당신은 전문 타로 리더입니다. 켈틱 크로스의 구조를 분석하여 카드들 간의 관계와 상호작용을 설명해주세요.

### 분석해야 할 구조

**1. 중심 십자가 (포지션 0-3)**:
- 현재 → 도전 → 과거 → 미래로 이어지는 시간적 흐름
- 이 네 카드가 만들어내는 핵심 스토리

**2. 수직축 (포지션 4-5)**:
- 의식적 목표(위)와 무의식적 영향(아래) 사이의 관계
- 내적 갈등이나 조화

**3. 수평선 (포지션 6-7)**:
- 조언과 외부 요인의 상호작용
- 실천 가능한 방향

**4. 감정과 결과 (포지션 8-9)**:
- 희망/두려움이 최종 결과에 미치는 영향
- 감정적 상태가 결과를 어떻게 형성하는지

**5. 전체 패턴**:
- 카드들 사이의 에너지 흐름
- 강조되는 아르카나나 수트의 의미
- 역방향 카드가 많다면 내적 도전
- 메이저 아르카나가 많다면 운명적 요소

**작성 가이드**:
- 각 구조적 그룹의 의미를 명확히 설명하세요
- 카드들 간의 연결고리와 상호작용을 분석하세요
- 전체 패턴이 질문자의 질문에 대해 무엇을 말하는지 종합하세요
- 300-500자 정도로 상세하게 작성하세요

## JSON 출력 형식

**필수**: 모든 응답은 다음 JSON 형식으로만 제공해주세요.

**JSON 문자열 규칙**:
- 모든 문자열 값 내부의 줄바꿈은 반드시 `\n`으로 이스케이프해야 합니다
- 실제 개행문자(Enter)를 사용하지 말고, 문자열 리터럴 `\n`을 사용하세요
- 탭은 `\t`로, 따옴표는 `\"`로 이스케이프하세요
- 모든 문자열은 반드시 한 줄로 작성하세요

```json
{
  "card_relationships": "열 장의 카드가 함께 만들어내는 구조적 관계와 패턴 설명. 중심 십자가, 수직축, 수평선, 감정과 결과의 상호작용을 포함 (300-500자, 문자열 내부에 실제 개행문자 없이 작성)"
}
```

**다시 한번 강조**: JSON 문자열 값 안에 실제 엔터키로 줄바꿈하지 마세요. 모든 텍스트는 한 줄로 작성하거나, 필요시 `\n` 이스케이프 시퀀스를 사용하세요.
