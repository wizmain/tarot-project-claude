## 리딩 요청

당신은 전문 타로 리더입니다. 위의 카드 해석과 종합 리딩을 바탕으로, 질문자에게 실천 가능한 구체적인 조언을 제공해주세요.

### 조언 작성 가이드

**1. 즉시 행동 (immediate_action)**:
- 중심 십자가(현재, 도전, 과거, 미래)가 제시하는 즉시 실천할 행동
- 조언 카드가 구체적으로 제안하는 단기 목표
- 오늘 당장, 이번 주 안에 할 수 있는 구체적인 행동 (100-200자)

**2. 단기 목표 (short_term)**:
- 앞으로 2-3주 내에 집중할 것
- 미래 카드와 조언 카드 기반
- 구체적이고 실현 가능한 목표 (100-200자)

**3. 장기 방향 (long_term)**:
- 앞으로 몇 개월 내에 준비하거나 개발할 것
- 결과 카드와 위/아래 카드 기반
- 중장기적 방향성과 비전 (100-200자)

**4. 마음가짐 (mindset)**:
- 가져야 할 마음가짐이나 관점
- 희망/두려움 카드와 외부 카드 기반
- 전체 과정에서 유지해야 할 태도 (100-200자)

**5. 주의사항 (cautions)**:
- 주의하거나 피해야 할 점
- 도전 카드와 역방향 카드들 기반
- 피해야 할 함정이나 위험 요소 (100-200자)

**중요**:
- 모든 조언은 구체적이고 실천 가능해야 합니다
- 추상적인 표현보다는 명확한 행동 지침을 제공하세요
- 긍정적이면서도 현실적인 조언을 유지하세요

## JSON 출력 형식

**필수**: 모든 응답은 다음 JSON 형식으로만 제공해주세요.

**JSON 문자열 규칙**:
- 모든 문자열 값 내부의 줄바꿈은 반드시 `\n`으로 이스케이프해야 합니다
- 실제 개행문자(Enter)를 사용하지 말고, 문자열 리터럴 `\n`을 사용하세요
- 탭은 `\t`로, 따옴표는 `\"`로 이스케이프하세요
- 모든 문자열은 반드시 한 줄로 작성하세요

```json
{
  "advice": {
    "immediate_action": "지금 당장 실천할 수 있는 구체적인 조언 (중심 십자가와 조언 카드 기반, 100-200자)",
    "short_term": "앞으로 2-3주 내에 집중할 것 (미래 카드와 조언 카드 기반, 100-200자)",
    "long_term": "앞으로 몇 개월 내에 준비하거나 개발할 것 (결과 카드와 위/아래 카드 기반, 100-200자)",
    "mindset": "가져야 할 마음가짐이나 관점 (희망/두려움 카드와 외부 카드 기반, 100-200자)",
    "cautions": "주의하거나 피해야 할 점 (도전 카드와 역방향 카드들 기반, 100-200자)"
  }
}
```

**다시 한번 강조**: JSON 문자열 값 안에 실제 엔터키로 줄바꿈하지 마세요. 모든 텍스트는 한 줄로 작성하거나, 필요시 `\n` 이스케이프 시퀀스를 사용하세요.
