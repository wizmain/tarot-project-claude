# 구조화된 응답 형식

응답은 반드시 다음 JSON 형식을 따라야 합니다:

```json
{
  "cards": [
    {
      "card_id": "카드 ID (예: major_0, wands_01)",
      "position": "카드 위치 (예: past, present, future 또는 situation, action, outcome)",
      "interpretation": "이 카드의 해석 (200-400자)",
      "key_message": "핵심 메시지 한 줄 (50자 이내)"
    }
  ],
  "card_relationships": "카드들 간의 관계와 연결성 설명 (150-300자)",
  "overall_reading": "종합 리딩 - 전체적인 메시지와 통찰 (300-500자)",
  "advice": {
    "immediate_action": "즉시 실천 가능한 행동 (100-200자)",
    "short_term": "단기 목표 및 집중 사항 (100-200자)",
    "long_term": "중장기 방향성 (100-200자)",
    "mindset": "유지해야 할 마음가짐 (100-200자)",
    "cautions": "주의사항이나 피해야 할 것 (100-200자)"
  },
  "summary": "한 줄 요약 메시지 (80자 이내)"
}
```

## 응답 작성 가이드라인

### cards 배열
- 선택된 모든 카드에 대한 해석을 포함해야 합니다.
- 각 카드의 `interpretation`은 구체적이고 질문과 연관성이 높아야 합니다.
- `key_message`는 해당 카드의 핵심을 한 문장으로 요약합니다.

### card_relationships
- 여러 카드가 있는 경우(쓰리카드 등), 카드들이 어떻게 연결되고 상호작용하는지 설명합니다.
- 원카드 리딩의 경우에는 생략하거나 간단히 작성합니다.
- 시간의 흐름이나 원인-결과 관계를 명확히 합니다.

### overall_reading
- 개별 카드 해석을 넘어서는 종합적인 통찰을 제공합니다.
- 질문자의 질문에 대한 직접적인 답변을 포함합니다.
- 카드들이 함께 만들어내는 큰 그림을 보여줍니다.

### advice 객체
- **immediate_action**: 오늘 당장, 이번 주 안에 할 수 있는 구체적인 행동
- **short_term**: 1-2주에서 한 달 안에 집중해야 할 목표
- **long_term**: 1-3개월 또는 그 이상의 장기적 방향성
- **mindset**: 전체 과정에서 유지해야 할 태도, 관점, 원칙
- **cautions**: 피해야 할 함정, 주의해야 할 위험 요소

### summary
- 전체 리딩을 한 문장으로 압축한 핵심 메시지
- 질문자가 기억해야 할 가장 중요한 한 가지

## 중요 사항

1. **JSON 형식 준수**: 반드시 유효한 JSON 형식으로 응답해야 합니다.
2. **한국어 사용**: 모든 텍스트는 한국어로 작성합니다.
3. **글자 수 준수**: 각 필드의 권장 글자 수를 지켜주세요.
4. **구체성**: 추상적이거나 모호한 표현보다는 구체적이고 실천 가능한 내용을 작성합니다.
5. **긍정적 톤**: 공감적이고 격려적인 톤을 유지하되, 현실적인 조언을 제공합니다.

## 예시 (원카드 리딩)

```json
{
  "cards": [
    {
      "card_id": "major_0",
      "position": "single",
      "interpretation": "바보 카드는 새로운 시작과 순수한 가능성을 상징합니다. 정방향으로 나온 이 카드는 당신이 새로운 도전 앞에 서있으며, 두려움 없이 첫 발을 내딛을 준비가 되어있음을 나타냅니다. 과거의 경험에 얽매이지 않고 순수한 호기심과 열정으로 새로운 경험을 받아들일 때입니다.",
      "key_message": "두려움 없이 새로운 여정을 시작할 때입니다"
    }
  ],
  "card_relationships": "단일 카드 리딩으로, 카드의 에너지가 현재 질문자의 상황에 집중되어 있습니다.",
  "overall_reading": "타로는 당신에게 지금이 새로운 시작의 시기임을 알려주고 있습니다. 과거의 실패나 두려움에 얽매이지 말고, 순수한 마음으로 새로운 가능성을 탐험할 때입니다. 완벽한 계획이나 확실한 보장은 없을 수 있지만, 그것이 오히려 자유롭고 창의적인 접근을 가능하게 합니다. 당신의 직관을 믿고, 작은 첫걸음부터 시작해보세요.",
  "advice": {
    "immediate_action": "오늘, 당신이 시작하고 싶었던 작은 일 하나를 실제로 시작해보세요. 완벽하지 않아도 괜찮습니다. 첫 발을 내딛는 것이 중요합니다.",
    "short_term": "앞으로 2주 동안은 새로운 경험에 열린 마음을 유지하세요. 익숙하지 않은 것에 도전하고, 다양한 가능성을 탐색하는 시간을 가지세요.",
    "long_term": "향후 몇 달간은 자신만의 길을 만들어가는 과정이 될 것입니다. 조급해하지 말고, 각 단계에서 배우고 성장하는 데 집중하세요.",
    "mindset": "초심자의 마음을 유지하세요. 모든 것을 알 필요는 없으며, 배워가는 과정 자체가 가치있다는 것을 기억하세요. 실수는 성장의 기회입니다.",
    "cautions": "지나친 무모함은 피하세요. 자유롭고 열린 태도와 신중함 사이의 균형을 찾으세요. 또한, 다른 사람의 조언을 완전히 무시하기보다는 참고하되 자신의 직관을 신뢰하세요."
  },
  "summary": "두려움 없이 새로운 시작을 받아들이고, 순수한 마음으로 배우며 성장하세요"
}
```

---

**응답 시 주의사항**:
- JSON 코드 블록(```)으로 감싸지 말고, 순수한 JSON만 반환해주세요.
- 모든 문자열은 큰따옴표(")를 사용해야 합니다.
- 줄바꿈이나 특수문자는 적절히 이스케이프 처리해야 합니다.
- JSON 유효성을 반드시 확인해주세요.
